# 💡 アイデア名
**「ステータスボード」— 今誰が何してる？リアルタイム稼働状況ダッシュボード**

## 👥 ターゲットユーザー
- リモートワーク中心で「今あの人に話しかけていいか分からない」と感じている社員
- チームの稼働状況を把握したいリーダー・マネージャー

## 😤 課題
- リモートワークだと、同僚が今忙しいのか暇なのか分からない
- 「ちょっといいですか？」の一言を送るのに心理的ハードルがある
- チーム全体の稼働感が見えず、仕事の偏りに気づけない

## 💊 解決策
各メンバーがワンクリックで自分のステータス（集中中・相談OK・離席中など）を更新し、チーム全体の状況がダッシュボードで一目で分かるツール。

## 🏗️ 主要機能（3つまで）
1. **ステータス更新** — 「集中中」「相談OK」「離席」などをワンクリック切り替え
2. **チームダッシュボード** — メンバー全員の現在ステータスをリアルタイム表示
3. **話しかけボタン** — 「相談OK」の人にワンクリックでSlack DMを開ける

## ⚡ 技術的な実現可能性
- 使用技術: React + Firebase Realtime Database
- 難易度: ★☆☆
- ハッカソン内で完成可能か: **Yes** — Firebaseのリアルタイム同期を使えば、最小限のコードでリアルタイム更新が実現できる

## 🎯 審査員へのアピールポイント
技術的にはシンプルだが、リモートワークの「声をかけづらい」という普遍的な課題を解決する。導入ハードルが極めて低い点も強み。
