# 💡 アイデア名
**「アサインくん」— 会議のタスク自動振り分けボット**

## 👥 ターゲットユーザー
- 会議後に「で、誰がやるんだっけ？」となりがちなプロジェクトチーム
- 議事録係の負担が大きいと感じているマネージャー層

## 😤 課題
- 会議で決まったタスクが、担当者に正しく伝わらない
- 議事録からタスクを手動で抜き出してチケット化する手間が大きい
- 会議後のフォローアップが属人的で漏れが発生する

## 💊 解決策
会議の議事録テキスト（またはメモ）を貼り付けると、AIがタスク・担当者・期限を自動抽出し、一覧化してくれるWebアプリ。ワンクリックでSlack通知やCSVエクスポートも可能。

## 🏗️ 主要機能（3つまで）
1. **テキスト解析** — 議事録からタスク・担当者・期限をAIで抽出
2. **タスク一覧表示** — 抽出結果を編集可能なテーブルで表示
3. **Slack通知** — 担当者にタスクをDMで自動通知

## ⚡ 技術的な実現可能性
- 使用技術: Next.js + OpenAI API + Slack Webhook
- 難易度: ★★☆
- ハッカソン内で完成可能か: **Yes** — AIのプロンプト設計がキモだが、UI自体はシンプル。Slack連携もWebhookで手軽に実装可能

## 🎯 審査員へのアピールポイント
「会議のあとの面倒」という誰もが共感する課題を、AI活用でスマートに解決する実用性の高さ。
